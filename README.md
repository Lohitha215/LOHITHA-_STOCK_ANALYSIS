# LOHITHA-_STOCK_ANALYSIS

Project Description:
                This project analyzes historical stock market data using Python and MySQL 
to identify trends, calculate returns, and visualize performance.

Objective:
        - Analyze historical stock prices
        - Calculate daily returns
        - Identify trends
        - Visualize stock performance

Technologies Used:
         - Python
         - Pandas
         - Matplotlib
         - MySQL
        - Jupyter Notebook
